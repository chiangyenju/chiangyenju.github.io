# Project Structure for Figma Showcases

## Recommended folder structure:
```
public/projects/figma-projects/
├── mobile-banking-app/
│   ├── hero-mockup.jpg          # Main showcase image
│   ├── screens/                 # Individual screen designs
│   │   ├── onboarding-1.jpg
│   │   ├── dashboard.jpg
│   │   ├── transaction-flow.jpg
│   │   └── profile.jpg
│   ├── typography/              # Typography showcases
│   │   ├── font-hierarchy.jpg
│   │   └── text-styles.jpg
│   ├── colors/                  # Color palette
│   │   ├── primary-palette.jpg
│   │   └── semantic-colors.jpg
│   ├── components/              # Design system components
│   │   ├── buttons.jpg
│   │   ├── forms.jpg
│   │   └── cards.jpg
│   └── user-flow/              # User flow diagrams
│       ├── onboarding-flow.jpg
│       └── payment-flow.jpg
├── ecommerce-web/
│   └── [same structure]
└── design-system/
    └── [same structure]
```

## Asset Export Guidelines:
- Hero images: 1200x800px (3:2 ratio)
- Screen mockups: 375x812px for mobile, 1440x900px for web
- Typography: 800x600px
- Color palettes: 600x400px
- Components: Variable sizes, maintain aspect ratios
- User flows: Export as wide images, 1600x variable height
